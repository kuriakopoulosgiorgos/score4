## Score 4

An online game for the score4 board game. The project is using [Microsoft Orleans](https://learn.microsoft.com/en-us/dotnet/orleans/overview)
for the backend and Web Components for the front end.

The game logic is placed in the Domain project, SignalR is used to propagate game updates to the connected players and provide
realtime data to the front end.
